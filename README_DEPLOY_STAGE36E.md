## What changed in Stage 36E

- Adds editor-level Undo and Redo buttons to the slide action toolbars.
- Adds keyboard shortcuts:
  - Undo: `Ctrl/⌘ + Z`
  - Redo: `Ctrl/⌘ + Shift + Z` or `Ctrl/⌘ + Y`
- Tracks deck/editor mutations including slide changes, block edits, imports, theme edits, preset changes, and figure edits.
- Preserves Stage 36D generated-HTML toolbar layout and Stage 36C pointer option behavior.
