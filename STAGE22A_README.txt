  js/commands.js

Shortcuts:
  Ctrl/Cmd+S: save current slide to deck/autosave
  Alt+Shift+P: refresh preview
  Alt+Shift+N: add slide
  Alt+Shift+U: update slide
  Alt+Shift+D: duplicate slide
  Alt+Shift+ArrowLeft/Right: previous/next slide
  Alt+Shift+ArrowUp/Down: move slide up/down
  Alt+Shift+B: add block
  Alt+Shift+E: update selected block
  Alt+Shift+X: delete selected block
  Escape: close figure modal if open

The Alt+Shift shortcuts are ignored while typing in text fields.
