# Stage 37G Release Notes

Stage 37G performs the UI reset requested after the Stage 36 feature expansion. It keeps the existing editor/runtime behavior while adding a calmer, task-oriented authoring shell.

## Included UI reset passes

- **37A information architecture:** top-level Create / Edit / Present / Advanced workflow bar.
- **37B contextual inspector:** clearer selected-item inspector state for slide/text/title/figure selections.
- **37C Copilot wizard polish:** prompt/spec/reference/settings are grouped as a guided generation workflow.
- **37D export workspace:** Present mode emphasizes generated-HTML controls and export actions.
- **37E visual polish:** softer panels, calmer spacing, reduced always-visible clutter, diagnostics hook, and internal UI smoke status.

## Preserved from prior stages

- Copilot deck spec / outline generation.
- Reference uploads, URLs, and PDF reference inputs.
- Image generation and custom HTML demo blocks.
- Real font-size support.
- Notebook and Chalkboard themes.
- Generated presentation controls and navigation fixes.
