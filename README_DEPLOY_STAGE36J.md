# Lumina Presenter Stage 36J Deploy Package

Stage 36J is a hotfix over Stage 36I.

Fixes:
- Restores generated HTML opening by escaping the slide-join newline inside the generated viewer script.
- Keeps optional generated controls in the fixed top-right toolbar, separate from Previous/Next.
- Makes export-controls dropdown option labels robust with explicit data labels and CSS fallback rendering.

Deploy by uploading this folder's contents to the static hosting root.
