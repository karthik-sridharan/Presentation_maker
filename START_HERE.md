## 🐛 Troubleshooting

**Problem:** "The UI doesn't work on GitHub Pages"

**Solution:** Make sure you deployed `index-working.html`, not `index.html`

**Problem:** "Buttons are dummy/don't respond"

**Solution:** You deployed the modular version (`index.html`) which is incomplete. Use `index-working.html` instead.

**Problem:** "Page is blank"

**Solution:** Check browser console (F12) for errors. Ensure the file is at your repo root.
