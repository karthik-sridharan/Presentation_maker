# Release Notes — Stage 38K

## Compact toolbar + preview lift

The top Keynote-style toolbar was visually too tall. Stage 38K compresses the toolbar and buttons so the preview moves upward and the app opens with a cleaner canvas-first layout.

## Diagnostics

Adds `stage38KCompactToolbarStatus` and `stage38KCompactToolbarDiagnostics` with measured toolbar height, button height, row gap, preview wrap padding, and preview frame position.
