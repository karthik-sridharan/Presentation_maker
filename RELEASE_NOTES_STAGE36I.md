# Release Notes — Stage 36I

Stage 36I fixes two regressions reported after Stage 36H:

- Generated HTML optional controls are now rendered once in a fixed top-right toolbar, with reserved space so they do not overlap the fixed Previous/Next buttons on the left.
- The export-controls dropdown options now wrap each label in an explicit visible text span so checkbox option text remains visible and readable.

Previous/Next remain always included in generated HTML.
