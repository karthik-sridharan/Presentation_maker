# Release Notes — Stage 36H

Stage 36H fixes two UI regressions from the generated HTML controls work:

- Right-aligns optional generated-presentation controls so they do not overlap the fixed Previous/Next toolbar.
- Keeps optional controls on the top-right on normal screens, with a small-screen fallback below the nav buttons.
- Replaces the export controls dropdown summary with a compact selected-controls display.
- Keeps the checkbox list functional and updates the summary immediately as choices change.
- Preserves Stage 36G export-time control capture, Stage 36E undo/redo, and Stage 36B touch double-tap navigation.
